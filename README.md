**LISTADO DE TEMAS**

1. **Modal y POP - UP**

Son ventanas emergentes que se pueden utilizar para mostrar información a los usuarios, sin embargo, un modal es una ventana emergente que bloquea la interacción con el resto de la interfaz, en cambio un Pop – Up es una ventana emergente que se abre de forma independiente y no siempre requiere interacción de parte del usuario.
